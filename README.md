# TwitterCleaner

## 한국어

X(Twitter)에서 내가 작성한 게시물 삭제 및 재게시(리트윗) 취소를 도와주는 Chrome 확장 프로그램입니다.

공식 X API를 사용하지 않고, 브라우저 화면의 버튼을 직접 조작하는 방식으로 동작합니다.

### 기능

- 내가 작성한 게시물 삭제
- 내가 재게시한 게시물 재게시 취소
- 현재 화면에 보이는 게시물 기준 처리
- Chrome 확장 프로그램 기반
- API Key 및 서버 불필요

### 설치 방법

1. 이 저장소를 다운로드합니다.
2. Chrome에서 `chrome://extensions` 접속
3. 우측 상단 `개발자 모드` 활성화
4. `압축해제된 확장 프로그램 로드`
5. 프로젝트 폴더 선택

### 사용 방법

1. X 로그인
2. 본인 프로필 페이지 이동
3. 확장 프로그램 실행
4. `리트윗 청소 시작` 버튼 클릭

### 주의사항

- X UI 변경 시 동작하지 않을 수 있습니다.
- 삭제 작업은 되돌리기 어렵습니다.
- 과도한 반복 실행은 계정 제한의 원인이 될 수 있습니다.
- 사용자는 본인의 책임하에 사용해야 합니다.

## English

Chrome extension for cleaning reposts and deleting your own posts on X (Twitter).

This project does not use the official X API.  
Instead, it directly interacts with buttons on the browser page.

### Features

- Delete your own posts
- Undo reposts
- Works on currently visible posts
- No API Key required
- No backend server required

### Installation

1. Download this repository
2. Open `chrome://extensions`
3. Enable `Developer Mode`
4. Click `Load unpacked`
5. Select the project folder

### Usage

1. Login to X
2. Open your profile page
3. Open the extension
4. Click `Start Cleaning`

### Warning

- May stop working if X changes its UI
- Deleted posts cannot be easily restored
- Excessive automation may risk account restriction
- Use at your own responsibility

## 日本語

X（Twitter）の再投稿取り消しおよび自分の投稿削除を支援するChrome拡張機能です。

公式X APIは使用せず、ブラウザ画面上のボタンを直接操作して動作します。

### 主な機能

- 自分の投稿削除
- 再投稿取り消し
- 現在画面に表示されている投稿を対象に処理
- Chrome拡張機能ベース
- API Key不要
- サーバー不要

### インストール方法

1. このリポジトリをダウンロード
2. Chromeで `chrome://extensions` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」
5. プロジェクトフォルダを選択

### 使用方法

1. Xにログイン
2. 自分のプロフィールページを開く
3. 拡張機能を実行
4. 「リツイート掃除開始」をクリック

### 注意事項

- XのUI変更により動作しなくなる場合があります
- 削除操作は元に戻せない場合があります
- 過度な自動化はアカウント制限の原因となる可能性があります
- 自己責任でご使用ください
